# table

A library for Dart developers. It is awesome.

## Usage

Create a table with two columns
```dart
Table t = new Table()
  ..addColumn(['BWI', 'BWI', 'BOS'], name: 'code')
  ..addColumn([1, 2, 3], name: 'value');
```
